# l3m-completion

> Generate shell completion scripts for l3m commands.
> More information: <https://github.com/mo/l3m-backend>.

- Generate bash completion script:

`l3m-completion bash`

- Generate zsh completion script:

`l3m-completion zsh`

- Generate fish completion script:

`l3m-completion fish`

- Enable bash completion (add to ~/.bashrc):

`source <(l3m-completion bash)`

- Enable zsh completion (add to ~/.zshrc):

`source <(l3m-completion zsh)`
